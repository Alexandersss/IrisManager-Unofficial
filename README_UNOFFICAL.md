Iris Manager U ((Unofficial))
=============================
it
    Iris Manager "U" è un progetto non ufficiale della versione liscia dell'iris manager ( Estwald ).
    Con questa versione, vogliamo portare avanti il progetto continuato da ESTWALD, ma che ( per motivi ovvi )
    non viene più frequentemente aggiornato con i vary payload/bugfix.
	Su questa versione, verrà introdotto solamente il minimo indispensabile per poter essere compatibile con futuri 
	CFW, lasciando pulita la sua interfaccia senza stravolgere nulla, ecco perche' il nome di "Iris Manager U"
    Ed è qui che entriamo in ballo noi.
	
eng  
    Iris Manager "U" is an unofficial project manager of the smooth version of the iris (Estwald).
    With this release, we want to take the project forward to continued Estwald, but that (for obvious reasons)
    does not vary more frequently updated with the payload / bugfixes.
	On this version, you will be introduced only the bare minimum in order to be compatible with future
	CFW, leaving its clean interface without changing anything, that 's why the name "Iris Manager U"
    And that's where we come into play us.
